# [remets](http://remets.herokuapp.com)

Handing over documents, made simple.

## Getting started

```sh
brew install pg
bundle exec rake db:create db:migrate
bundle exec rails server # go to localhost:3000

# To load secrets
source ./some-folder-not-in-the-repo/secrets.sh
```
